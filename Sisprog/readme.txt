O código fonte no arquivo LD.txt pode ser executado da seguinte forma:
1- mude o nome para LD.c
2- compile o código da melhor forma para você (o projeto foi desenvolvido sobre o gcc 9.3 em um ubuntu 20.04 LTS)
3- execute o código

Em caso de erro com o nome dos arquivos txt copie o nome do arquivo e cole no código, na minha maquina as vezes dizia que o arquivo não existia.

O código fonte foi enviado em formato txt para que outro leitores de texto pudessem abrir adequadamento
